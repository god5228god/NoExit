# NoExit — 방탈출 파티 매칭 플랫폼

방탈출을 함께 즐길 팀원을 먼저 모으고, 팀이 완성되면 테마를 예약하는 파티 매칭 서비스

<br>

## 프로젝트 정보

| | |
|---|---|
| 기간 | 2026.05.25 ~ 2026.06.13 (3주) |
| 인원 | 4인 팀 프로젝트 |
| 역할 | 팀장 · 예약 파트 |
| 스택 | Java · Spring Boot · Oracle · MyBatis · JSP · jQuery |

<br>

## 내 담당 구현 범위

**카페 관계자**
- 예약 슬롯 등록 / 비활성화 (`/owner/openRes`)
- 예약 현황 조회 — 날짜·카페·테마 필터 + 더보기 페이징 (`/owner/resList`)
- 예약 강제 취소

**일반 사용자**
- 파티장 예약 확정 처리 (`/reservation`)

**DB**
- 예약 관련 뷰 · 함수 · 프로시저 설계 및 구현
